Part B: True / False

Statement Answer
Every link inside <ul> must be placed inside an <li>. True
A page may have multiple <h1> headings to make text larger. False
aria-current="page" should be used on the active navigation item. True
A form is accessible even if it has no <label> elements. False
KLM predicts execution time for expert, error-free performance. True

Part C: Short Answer

Difference Between Semantic and Generic Elements

Semantic elements like <nav> describe their meaning clearly, while generic elements like <div> are used only for grouping or styling.

Why KLM Does Not Model High-Level Planning and Interpretation

KLM predicts execution time for routine expert actions but does not model high-level cognitive processes because they vary between users and are difficult to measure.
